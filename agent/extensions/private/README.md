# Private Pi Extensions

Extensions in this directory are for this user's local Pi workflow.

Private extensions may encode local preferences, installed-tool assumptions, account-specific compatibility notes, or unpublished workflow behavior. They should still avoid secrets and accidental machine-specific paths unless those paths are intentionally documented as local-only.

Do not publish packages from this directory as-is. If a private extension becomes generally useful, first remove personal assumptions, document dependencies and fallback behavior, move it to `../public/`, and update the root extension manifest.

There are currently no private extension packages.
