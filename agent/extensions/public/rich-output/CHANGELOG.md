# Changelog

## 0.1.0

- Initial public package release.
- Added terminal-friendly timeline cards for structured summaries, findings, validation results, diagrams, charts, artifact links, status blocks, and inline image previews.
- Added safe fallback behavior for terminals without image/link support and for unavailable optional render tools.

This extension is working but still WIP. I am publishing it while evaluating whether rich timeline output is useful enough to keep and expand.
