# Execute Long Plan

This is a compatibility skill. The primary execution skill is `execute-plan`.

When this skill is selected:
1. Read `../execute-plan/SKILL.md`.
2. Read `../execute-plan/long-plan.md`.
3. Execute using `execute-plan` behavior for a bounded split/long plan.

Do not use this skill for open-ended or unbounded work. For open-ended loops, use `execute-plan` and read `../execute-plan/unbounded-work.md`.
