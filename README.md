# Personal Pi Setup

This repository contains my personal [Pi](https://pi.dev) agent setup: global instructions, skills, extensions, keybindings, and local workflow glue.

It is published as a reference, not a polished distribution. Feel free to copy, adapt, or ignore anything that is useful to you. Some files are intentionally tuned for my own workflow and may assume my preferences or local setup.

## Public extensions

The public, reusable Pi extensions live in:

```text
agent/extensions/public/
```

Current public extensions:

- `@badliveware/pi-compaction-continue`
- `@badliveware/pi-pr-upstream-status`
- `@badliveware/pi-footer-framework`
- `@badliveware/pi-model-catalog`

Each extension is packaged independently from its own directory. Publishing notes are in:

```text
agent/extensions/PUBLISHING.md
```

## Private/local pieces

Everything else should be treated as personal configuration unless stated otherwise. It may still be useful as an example, but it is not promised to be stable or generally applicable.
